# Overview

This repository contains Verilog HDL implementations of fundamental digital logic circuits created while learning RTL Design, Digital Electronics, and VLSI concepts using Xilinx Vivado.

The repository focuses on:
- Understanding combinational logic design
- Writing synthesizable Verilog HDL
- Simulating digital circuits
- Analyzing simulation waveforms
- Building a strong foundation for VLSI and RTL design
